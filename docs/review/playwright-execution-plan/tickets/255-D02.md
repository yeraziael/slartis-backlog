The final Gitea evidence comment MUST contain:

- `instruction_file`, `instruction_commit`, `instruction_reloaded`, `pr_head_commit`, `changed_files`, `ci_run_ids`, and `review_target: pr` exactly as required by `Homelab/Architecture/AGENTS.md`.
- Base commit, branch, Gitea PR URL, exact test commands and outcomes, required CI job names/run URLs, and an acceptance-criterion-to-evidence matrix.
- Declaration of whether runtime, deployment, DNS, secrets, accounts, containers or external systems were mutated.
- Rollback statement and the complete findings record.
- Transport validation: PR head equals the evidenced SHA; changed files equal the permitted scope; CI and evidence links resolve for the reviewer.

Approval is valid only for the evidenced head SHA. A later push invalidates it and requires re-review.
