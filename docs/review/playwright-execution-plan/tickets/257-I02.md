## In Scope

- Add deterministic local-page self-tests for browser startup and two-context state isolation.
- Make self-tests run through the standard platform target inside the ephemeral container.

## Explicit Non-Goals

- No external network, Homelab service, authentication, evidence generation or result remapping.

## Prerequisites

- PW-I01 merged; pinned runner image is pullable in the test environment.

## Normative Implementation Contract

- Tests MUST use controlled local content and MUST NOT depend on Internet or Homelab availability.
- The second context MUST prove cookies, local storage and session storage from the first are absent.
- Runner container MUST be removed after both pass and fail paths.

## Acceptance Criteria

- Chromium startup test passes.
- Isolation test fails if any browser state leaks.
- No container or writable browser state remains after the target exits.

## Required Tests

- `make test-playwright-platform`
- Run the established runner once with a forced assertion failure and verify container cleanup.

- The required Gitea Actions jobs for the PR head MUST be green. An absent job is an evidence deficiency, not a pass.
