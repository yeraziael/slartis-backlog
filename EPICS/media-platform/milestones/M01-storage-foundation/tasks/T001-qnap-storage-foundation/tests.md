# Tests

- Mounts ueberstehen Host-Neustart und kurzzeitigen NAS-Ausfall.
- Container erkennen einen fehlenden Mount statt ein leeres lokales Verzeichnis zu verwenden.
- Jeder Medientyp ist getrennt zugreifbar.
- Testdateien koennen aus `incoming` atomar in einen Zielbereich verschoben werden.
- Gleichzeitige strukturveraendernde Testjobs werden serialisiert.
- Rechte entsprechen der dokumentierten Matrix.
