# Tests

- Einzelvideo wird genau einmal heruntergeladen.
- Kanalabo erkennt nur neue Uploads; leerer Scan erzeugt keine Datei.
- Abbruch hinterlaesst keine halbfertige Zieldatei.
- Ein bestehender Film wird analysiert, Jellyfin-konform umbenannt, verschoben und indiziert.
- Niedrige Qualitaet wird nur markiert.
- Der Prozess kann ueber Tage pausiert und fortgesetzt werden.
