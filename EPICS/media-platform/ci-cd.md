# CI/CD

- Compose-Dateien werden syntaktisch validiert.
- Shell- und Konfigurationsdateien werden gelintet.
- Automatisierte Tests duerfen keine produktiven Medien veraendern.
- Integrationstests verwenden temporaere Testfreigaben und kleine Testdateien.
- Deployments erfolgen ueber den bestehenden Homelab-Workflow.
- Secrets kommen ausschliesslich aus der bestehenden Secret-Verwaltung.
