# Contract

- Lege einen Mirror des offiziellen Upstreams an.
- Lege einen separaten Fork aus einem festgehaltenen Upstream-Commit an.
- Konfiguriere Upstream als lesbares Remote im Fork.
- Fuege eine kurze MAINTENANCE.md mit Fetch-, Merge-/Rebase-, Konflikt-, Release- und Rollback-Verfahren hinzu.
- Keine Secrets, Volumes oder produktive CWA-Konfiguration committen.
