# Definition of Done

- Alle drei Milestones sind per Pull Request umgesetzt.
- CWA laeuft nach Upgrade mit bestehender Bibliothek und Persistenz.
- OIDC und die mobile Admin-Korrektur sind dokumentiert und getestet.
- Upstream-Sync-Anleitung, Rollback und Backup-Hinweise liegen vor.
